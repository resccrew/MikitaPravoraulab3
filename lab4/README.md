# Lab4 — Библиотека (Python)

Простой консольный проект для управления библиотекой: добавление книг, их выдача и возврат.

## Структура проекта

- `library.py` — логика доменной модели (классы `Book` и `Library`).
- `main.py` — точка входа, консольный интерфейс.
- `tests/` — модульные тесты для `library.py`.
- `requirements.txt` — зависимости проекта (в данном случае не требуются).
- `.gitignore` — стандартные исключения для Python-проекта.

## Запуск

1. Убедитесь, что установлен Python 3.10+ (у вас: 3.13.2).
2. (Опционально) создайте виртуальное окружение:
   - macOS/Linux: `python3 -m venv .venv && source .venv/bin/activate`
   - Windows (PowerShell): `python -m venv .venv; .venv\\Scripts\\Activate.ps1`
3. Установите зависимости (если появятся): `pip install -r requirements.txt`
4. Запустите программу: `python3 main.py`

## Тесты

Выполнить модульные тесты:

```bash
python3 -m unittest -q
```

## Примечания

- Проект не требует внешних библиотек; весь код — в стандартном Python.
- При необходимости можно оформить как пакет и добавить `pyproject.toml`/`setup.cfg`.
